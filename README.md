# UinvolveX

The official portfolio website for UinvolveX, an independent technology studio based in Guwahati, India.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

Pushing to `main` deploys the site through GitHub Actions. To use a custom domain, add it in the repository's **Settings → Pages** and configure the matching DNS records in Hostinger.
